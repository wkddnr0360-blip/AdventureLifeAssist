# 변경 내역

## 2.0.0 — 2026-07-23

### 추가

- 실시간 접속·공부 상태와 오늘·주간·누적 랭킹
- 사용자별 랭킹·현재 과목·온라인 공개 범위
- 친구 검색·요청·수락·거절·삭제
- 친구 전용 DM, 실시간 메시지, 읽음 상태
- 학습 로그·메모·할 일 통합 월간 캘린더와 어제/오늘 바로가기
- 일일 집중시간·문제 수 목표, 진행률, 업적, 달성 축하·알림
- 검증된 지정 이메일 또는 custom claim 기반 관리자 콘솔
- Realtime Database `onDisconnect` 접속 감지와 Firestore heartbeat 대체 경로
- 멀티기기 활성 세션 트랜잭션 검증
- 6단계 온보딩 튜토리얼
- 1~16 숫자 순서 집중력 리프레시 게임
- iOS safe-area·Apple touch icon·maskable icon·Windows 설치 대응
- 배포 구성, 관리자 claim 도구, 데이터 모델·보안·검증 문서

### 개선

- 모바일 텍스트 줄바꿈, 16px 입력 폰트, 가로 오버플로 방지
- 키보드 포커스 표시, 모션 감소 설정 대응
- 홈 빠른 실행, 하단 내비게이션, 카드·모달·랭킹·상태 UI
- 서비스 워커 캐시 버전과 이전 AdventureLifeAssist 캐시 정리 범위
- 라운지 좋아요·댓글·신고 규칙

### 유지

- 기존 `apps/focusbell_v1` 네임스페이스
- 기존 알람·학습 로그·프로필 데이터
- 이메일 로그인, 라운지, CSV/JSON 내보내기, 테마, PWA 기본 기능
